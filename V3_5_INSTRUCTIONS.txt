TGTC V3.5 - Stable Job URLs
===========================

What changed
------------
- Google Jobs detail-viewer URLs are never saved as Airtable Job URL.
- URL priority is now:
  1) employer careers page
  2) recognized ATS
  3) LinkedIn / Indeed
  4) other direct publisher
  5) working aggregator
- Known Google Jobs tracking parameters are removed from selected URLs.
- If Google Jobs is the only URL available, Job URL is left blank rather than
  saving a misleading link.
- Existing Google Jobs links can be repaired from the locally saved JSearch
  payload without re-running the full pipeline.

Install
-------
1. Copy all files from this patch into:
   C:\TGTC\tgtc_pipeline
2. Choose "Replace the files in the destination".

Validate (CMD)
--------------
cd /d C:\TGTC\tgtc_pipeline
python -m unittest discover -s tests -v

Expected:
Ran 37 tests
OK

Repair existing Airtable Google Jobs URLs
-----------------------------------------
python repair_airtable_job_urls_v35.py

The repair changes only:
- Job URL
- Job URL Status
- Job URL Source
- Job Signal Notes

It does not change Status, Relevance, contacts, Role Focus, or campaign fields.
No new Airtable columns are required.
