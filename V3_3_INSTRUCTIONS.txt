WHAT CHANGED
1. Working aggregator URLs are no longer treated as a second approval gate.
   - Job URL Source remains "aggregator" for transparency.
   - A URL that returns 2xx/3xx becomes Job URL Status="verified".
   - Legacy aggregator_review values no longer block Instantly.
   - Broken or genuinely unverified URLs remain blocked.

2. Relevance Score is now 0-100 in Airtable.
   Internal classifier points remain 0-8 for logic, but presentation is:
   1=13, 2=25, 3=38, 4=50, 5=63, 6=75, 7=88, 8=100.

3. Freshness checks all JSearch posted-date fields and conservatively uses
   the oldest one when the fields disagree.
